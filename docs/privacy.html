로또 번호 예상기(이하 "서비스")는 이용자의 개인정보를 중요시하며, 「개인정보 보호법」 및 관련 법령을 준수합니다. 본 개인정보처리방침은 서비스가 수집하는 개인정보의 항목, 이용 목적, 보관 기간, 이용자의 권리 등을 안내합니다.

1. 수집하는 개인정보 항목

1-1. 회원가입 시 (Google 소셜 로그인)

  • 필수: Google 계정 이메일 주소, 표시 이름, 프로필 사진 URL, 고유 사용자 ID (UID)
  • Google OAuth 2.0을 통해 제공되는 정보만 수집하며, Google 계정 비밀번호는 수집하지 않습니다.

1-2. 서비스 이용 시 자동 수집

  • 기기 정보: 브라우저 종류, 운영체제, 화면 해상도
  • 이용 기록: 접속 일시, 페이지 방문 기록, 번호 생성 횟수
  • 쿠키 및 로컬 저장소: 서비스 설정, 게임 진행 데이터 (클로버, 스트릭, 업적 등)

1-3. 번호 생성 시 입력 정보

  • 생년월일, 태어난 시간 (사주 기반 생성 시)
  • 이름 (이름 기반 생성 시)
  • MBTI 유형 (MBTI 기반 생성 시)

위 정보는 브라우저의 localStorage에만 저장되며, 서버로 전송되지 않습니다. 번호 생성 연산은 모두 사용자의 기기에서 수행됩니다.

2. 개인정보의 이용 목적

  • 회원 식별 및 로그인 상태 유지
  • 클라우드 데이터 동기화 (클로버 포인트, 업적, 즐겨찾기 등)
  • 서비스 이용 통계 분석 및 품질 개선
  • 이벤트 및 보상 제공

3. 개인정보의 보관 및 파기

  • 보관 기간: 회원 탈퇴 시까지 또는 마지막 로그인 후 1년
  • 파기 방법: Firebase Authentication 및 Firestore에서 해당 사용자 데이터를 삭제
  • localStorage 데이터: 사용자가 브라우저 데이터를 직접 삭제하여 제거 가능

4. 제3자 제공 및 위탁

서비스는 이용자의 개인정보를 제3자에게 판매하거나 제공하지 않습니다. 다만, 다음의 서비스를 이용합니다:

4-1. Firebase (Google)

  • 목적: 사용자 인증 (Authentication) 및 데이터 저장 (Firestore)
  • 처리 항목: 이메일, 표시 이름, UID, 게임 데이터
  • 개인정보처리방침: Google Privacy Policy 참조

4-2. Google Analytics 4 (GA4)

  • 목적: 서비스 이용 통계 분석
  • 처리 항목: 페이지 방문 기록, 이벤트 데이터 (비식별 처리)
  • IP 익명화: GA4는 기본적으로 IP 주소를 익명 처리합니다

4-3. Microsoft Clarity

  • 목적: 사용자 행동 분석 (히트맵, 세션 녹화)
  • 처리 항목: 마우스 움직임, 클릭 패턴, 스크롤 데이터
  • 개인 식별 정보는 자동으로 마스킹 처리됩니다

4-4. Kakao SDK

  • 목적: 카카오톡 공유 기능
  • 처리 항목: 공유 시 전달되는 URL 및 메시지 내용

5. 쿠키 및 추적 기술

서비스는 쿠키(Cookie)와 localStorage를 사용합니다.

  • 필수 쿠키: 로그인 세션 유지, 서비스 기본 기능 (비활성화 불가)
  • 분석 쿠키: GA4, Clarity에서 사용하는 분석용 쿠키 (브라우저 설정에서 차단 가능)
  • localStorage: 게임 데이터, 설정, 번호 생성 이력 저장 (브라우저에서 직접 삭제 가능)

6. 이용자의 권리

  • 정보 열람: 본인의 개인정보 처리 현황을 확인할 수 있습니다
  • 정보 수정: Google 계정 설정에서 이름, 프로필 사진을 수정할 수 있습니다
  • 회원 탈퇴: 서비스 내 설정에서 계정을 삭제할 수 있으며, 삭제 시 모든 클라우드 데이터가 제거됩니다
  • 데이터 삭제: 브라우저의 사이트 데이터 삭제를 통해 localStorage 데이터를 제거할 수 있습니다

7. 아동의 개인정보 보호

서비스는 만 14세 미만 아동의 개인정보를 의도적으로 수집하지 않습니다. 만 14세 미만 이용자의 개인정보가 수집된 것을 인지한 경우, 즉시 해당 정보를 삭제합니다.

8. 개인정보의 안전성 확보 조치

  • Firebase의 보안 규칙(Security Rules)을 통한 데이터 접근 제어
  • HTTPS 암호화 통신
  • 민감 정보(생년월일, 이름 등)는 서버에 저장하지 않고 클라이언트에서만 처리

9. 개인정보처리방침의 변경

본 방침이 변경되는 경우, 변경 사항을 서비스 내 공지사항 또는 이 페이지를 통해 안내합니다. 변경된 방침은 게시한 날부터 효력이 발생합니다.
